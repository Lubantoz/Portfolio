# Portfolio (Vite + React)

This is a minimal portfolio scaffold built with Vite and React.

Quick start

```bash
npm install
npm run dev
```

Build

```bash
npm run build
```

Deploy to GitHub Pages

1. Set the `repository` field in `package.json` to your repo URL.
2. Run:

```bash
npm run deploy
```

Replace placeholder content (name, bio, projects, contact email) in the files under `src`.
